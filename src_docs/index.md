## [Vexy Lines](https://vexy.art/lines/)

Vexy Lines transforms photos, illustrations, generated images and other bitmaps into expressive vector art. 

It reads the color value of every pixel and itelligently builds vector artwork from it. Feed it a portrait, get back an engraving. Drop in a landscape, receive flowing wave patterns. 

Vexy Lines has over a dozen ways to interpret your images: parallel lines that mimic copper-plate engravings, radiating strokes that explode from a center point, spirals that wind through your composition, fractals that fill space with mathematical precision. 

Each method responds to the light and dark areas of your source image differently.

## [Vexy Lines Run](https://vexy.dev/vexy-lines-run/)

Vexy Lines Run is a small desktop app that serves as an unofficial add-on to Vexy Lines. With the `vexy-lines-cli` app, you can: 

- Export multiple .lines files as SVG, PNG, JPG
- Apply one Vexy Lines style to multiple images or a video, and export the result as SVG, PNG, JPG, MP4

## [Vexy Lines API](https://vexy.dev/vexy-lines-apy/)

A pythonic wrapper for the Vexy Lines MCP API with additional functionality. 

## [Vexy Lines CLI/MCP](https://vexy.dev/vexy-lines-cli/)

Command-line app `vexy-lines-cli` and pythonic MCP server the for the Vexy Lines API. 

## [Vexy Lines Py](https://vexy.dev/vexy-lines-py/)

A pure-Python parser for the Vexy Lines .lines file format. Useful for inspection and programmatic manipulation of .lines files.
